# App Compliance Pages

Public pages for App Store submission metadata (privacy policy, support pages, etc.).

## Apps
- KeepIt

## KeepIt URLs
- Privacy Policy: `https://keviler.github.io/app-compliance-pages/keepit/privacy-policy.html`
- Support: `https://keviler.github.io/app-compliance-pages/keepit/support.html`

## KeepIt App Store Metadata
- zh-Hans metadata: `keepit/appstore-connect/zh-Hans-metadata.md`
- Submission checklist: `keepit/appstore-connect/submission-checklist.md`

## Reuse For New Apps
1. Copy `keepit/` to a new app folder.
2. Update app name, contact email, and policy content.
3. Add App Store metadata under `appstore-connect/`.
4. Commit and push to `main`, then use Pages URLs in App Store Connect.
